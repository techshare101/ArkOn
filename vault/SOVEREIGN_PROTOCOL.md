## 4. Hormozi Integration Rules

### 4.1 Value Equation (Applied to Knowledge)

```
VALUE = (Dream Outcome × Perceived Likelihood)
        ÷ (Time Delay × Effort/Sacrifice)
```

**For every note ask:**
- Does this increase dream outcome clarity?
- Does this increase perceived likelihood of achievement?
- Does this reduce time to outcome?
- Does this reduce effort required?

### 4.2 Offer Architecture (Applied to Skills)

Each skill in `02_Skills` must document:
1. **What it does** (dream outcome)
2. **Proof it works** (case studies, metrics)
3. **Speed to result** (implementation timeline)
4. **Ease of use** (complexity score 1-10)

### 4.3 Lead Generation (Applied to Federal Strike)

Every entry in `04_Federal_Strike` must include:
1. **Opportunity Score** (1-100)
2. **NAICS Match** (primary: 541690)
3. **Capability Gap** (what we need to win)
4. **Timeline** (submission deadline)
5. **Value** (contract ceiling)
