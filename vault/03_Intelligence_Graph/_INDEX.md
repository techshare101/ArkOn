# 🧠 Intelligence Graph

> Linked knowledge base for strategic advantage.

## Graph Nodes
- Research findings
- Market intelligence
- Technology assessments
- Competitive analysis
- Regulatory landscape

## Connection Types
- `relates-to` — thematic connection
- `depends-on` — prerequisite relationship
- `contradicts` — conflicting information
- `evolves-from` — temporal progression
- `enables` — capability unlock

## Weekly Expansion Protocol
1. Add 3+ new nodes per week
2. Create 5+ new links per week
3. Review orphan nodes monthly
4. Prune stale connections quarterly
